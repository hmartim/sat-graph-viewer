Ontological Parsimony. Every case is modelled with the same four primitives — Item, Version, Action, TextUnit — without new classes. Judicial interpretations create Interpretive Versions (I-TV) that overlay Statutory Versions (S-TV) without terminating them. Legislative amendment, judicial interpretation, overruling, and constitutional dialogue are expressed through topology alone. The Version Timeline view shows the raw getValidVersions(itemId, date) result that drives every panel on this page.
